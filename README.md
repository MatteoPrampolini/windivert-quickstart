# WinDivert Quickstart

This is a small quickstart for [Windivert](https://github.com/basil00/WinDivert).

WinDivert 2.2.2-A is shipped manually in this repo, with both 32-bit and 64-bit files.
There is no fancy download for the latest version.

The default build is x64, but x86 is already wired in the Makefile.

To run this project:

* `make && make run-cmd-admin`

Windivert requires admin privileges to run.
